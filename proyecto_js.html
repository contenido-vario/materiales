ACTIVIDAD
 

Proyecto final de Módulo

La actividad a realizar tiene dos partes:

  1. Dashboard / Menú de Ejercicios

    Realizar un Dashboard / Menu que enlace con todos los ejercicios de JavaScript que hemos realizado.

    Revisaré vuestro Dashboard desde el móvil. Tanto el Dashboard como los ejercicios a partir de Ejercicios condicionales deberán ser responsivos.

    Puntuación: 3 puntos (1 punto responsividad y 2 puntos entrega de todos los ejercicios.)

  2. Carrusel (con array de Objetos)

    Hacer un carrusel que muestre el contenido de un array de objetos el cual tendrá la siguiente estructura (ejemplo):

    {
      nombre: "Barcelona",
      descripcion: "Capital de Catalunya",
      imagen: "barcelona.png",
      caracteristicas:[
        "población: 1.740.782 (2025)",
        "P.I.B: €112.000 millones (2024)",
        "superficie: 101,9 km2"
      ]
    }

    Podéis usar cualquier tema siempre y cuando el array de objetos tenga los atributos nombre, descripcion, imagen y el array de características. Al llegar al final o al inicio, debe dar la vuelta. Mínimo 10 imágenes.

    La pantalla del carrusel deberá mostrar todos los datos del objeto.

Deberéis subir las URL del Dashboard y del Carrusel al siguiente formulario
