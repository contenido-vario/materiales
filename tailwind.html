¡Bienvenidos a Tailwind!

Esta es una página de ejemplo en Tailwind que utiliza las clases de espaciado, tipografía y responsive design.

Para ver el contenido de cada clase en Visual Studio Code, instala la extensión Tailwind CSS Intellisense

Para agregar variables y estilos adicionales se agrega una sección de estilos en el head:
<style type="text/tailwindcss">
...
</style>

Diseño Rápido

Tailwind CSS es un framework de CSS "utility-first" (prioridad a la utilidad) de código abierto diseñado para maquetar sitios web rápidamente sin salir del HTML.

Permite diseñar sin cambiar de archivo, agilizando el desarrollo.

Responsive

Mobile-First: Enfocado en el diseño responsivo desde dispositivos móviles.

Controla cómo se ve en móvil y desktop con prefijos como md:. Ver Responsive design

La clase container establece un max-width (ancho máximo) que cambia según el tamaño de la pantalla del usuario (los breakpoints).

Moderno y optimizado

Flexbox y Grid facilitan la creación de layouts complejos.

Tailwind utiliza una herramienta similar a PurgeCSS para eliminar el CSS no utilizado, resultando en archivos finales muy ligeros.
