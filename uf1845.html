Proyecto UF1845. CONSUMIR API REST

Para este proyecto, selecciona una API de tu preferencia, preferiblemente que permita mostrar imágenes.

Deberás crear una aplicación que consuma la API elegida y que tenga las siguientes características:

  • Encabezado con título y dos botones: uno para cambiar el tema (oscuro/claro) y otro para cambiar la visualización de los datos (formatos lista / galería).
    Un ejemplo de este último botón lo tienen en Google Drive.
  • Los datos a mostrar en cada tipo de visualización de datos, dependerán de la API seleccionada.
  • Los botones del encabezado deberán guardar la selección en el Local Storage.
  • Se deberá mostrar en el header, footer o aside (como gustéis) un nav que permita ir cargando los datos de la API dependiendo de cómo haya sido diseñada: por rangos de id (como la PokéAPI), por páginas (como la API de los Simpsons) o por letras (iniciales)/países (como The MealDB)
  • Cada registro de datos deberá contener un enlace al detalle de los mismos el cual se mostrará en un dialog.

Entrega:

  1. Sólo deberán entregarse 2 archivos: un .html y un .js.
  2. Deberéis subir las URL del proyecto al siguiente formulario
